Git

Git'as yra populiariausia kodo versijavimo programa.

Git Komandos

  • git init - sukuria nauj1 git'o repozitoriją
  • git add - prideda nurodytus failus prie commitui paruoštų failų (staged). ( . - visi failai)
  • git status - patikrinti gito būseną
  • git commit -m "žinutė" - sukuria kodo versiją iš stage'intų failų ir prideda žinutę apie pakitimus (žinutę išlaikyti ne ilgesnę negu 50 simbolių)
  • git push - iš lokalios repozitorijos naujų commit'ų išsiuntimas į išorinę
  • git pull - iš išorinės repozitorijos commit'ų atsisiuntimas į lokalią
  • remote add origin http://... - prijungimas išorinės repozitorijos prie lokalios
  • git log - commit'ų sąrašas
  • git checkout branchName|commitID - pereiti į kitą kodo šaką arba versiją.
  • branch naujosSakosPavadinimas - sukuria naują šaką
